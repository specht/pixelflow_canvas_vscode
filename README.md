# Pixelflow Canvas

A virtual CRT for old school graphics programming in Visual Studio Code.

## Features

- 📺 Create a virtual screen to draw on!
- 🔍 Choose your own resolution!
- 🎨 Use RGB colors or use a palette!
- 🧟 `set_pixel` and `get_pixel` are back!
